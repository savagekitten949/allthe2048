allthe2048
==========

Updated index of all the 2048 variants currently out there!

# Contributing

Feel free to:
- Fork
- Add your or anyone else's version to `_data/variants.yml` in the format
    - url: example.github.io/2048/
    - name: Example 2048
    - authtor: Example Exampleson
- Open a pull request

Thanks! :)
